# Authentication (Kimlik Doğrulama)
- Bir kullanıcının kimliğimliğini doğruluma süreci
- Kullanıcı adı, şifresi gibi bilgilern alınır

# Authorization (Yetkilendirme)
- Kullanıcının sistemin kaynaklarına/işlevlerine/sayfalarına erişimine izin verme veya reddetme sürecidir.

# About Project: 
- This project was created using firebase, the login with google feature has been activated, you can also register with your email account and log in with this email account. If you forget your password, it will send you a password renewal email. 
Functions such as editing photo upload posts, adding and withdrawing likes, calculating the current number of posts have been added.

-Bu proje firebase kullanılarak oluşturuldu, google ile giriş yap özelliği aktifleştirildi, aynı zamanda email hesabınız ile kayıt olabilir ve bu mail hesabı ile giriş yapabilirsiniz. Şifrenizi unuttuğunuz taktirde size şifre yenileme emaili gönderecektir. 
Fotoğraf yükleme gönderileri düzenleme, beğeni atıp geri çekme, güncel gönderi sayısını hesaplama gibi fonksiyonlarda eklenmiştir.
